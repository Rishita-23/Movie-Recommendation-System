# 🎬 Movie Recommendation System

## 📌 Problem Statement

With thousands of movies available, users often struggle to find relevant content. This project builds a recommendation system to suggest similar movies based on user preferences.

## 📊 Dataset

(https://www.kaggle.com/datasets/dastgeerjutt/movies)

## ⚙️ Approach

* Data Cleaning & Preprocessing
* Feature Engineering:

  * Combined genres, cast, keywords into a single feature
* Vectorization using CountVectorizer
* Similarity Calculation using Cosine Similarity

## 🤖 Model

* Content-Based Filtering

## 📈 Output

* Recommends top 5 similar movies for a given input


## 🛠️ Tech Stack

* Python
* Pandas
* Scikit-learn

## 📌 Key Learning

* Understanding recommendation systems
* Feature engineering for text data
* Similarity-based modeling
